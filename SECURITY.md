# Security Policy

## Supported Versions
We support security fixes on the latest **main** branch until the next minor release.

## Reporting a Vulnerability
If you discover a vulnerability, please email **harounsmida@icloud.com** with:
- A description of the issue and potential impact
- Steps to reproduce (PoC if possible)
- Affected commit/version

Please allow up to **72 hours** for acknowledgement. We’ll coordinate a fix and credit you if desired.

## Scope
- App source code in this repository
- Build/CI workflows in `.github/workflows/`

Out of scope: third-party package vulnerabilities (report those upstream).

## Disclosure
We follow **coordinated disclosure**: we will merge a fix and release before public disclosure whenever feasible.
