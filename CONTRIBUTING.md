# Contributing to Habit Tracker

Thanks for your interest in contributing! 🎉

## How to contribute
1. Fork the repo and create your branch from `main`.
2. Make your changes (keep commits clear & descriptive).
3. Test on Android and iOS before opening a PR.
4. Submit a Pull Request and link related issues.

## Coding guidelines
- Use **Flutter stable** channel.
- Run `flutter analyze` and `flutter test` before pushing.
- Follow [Conventional Commits](https://www.conventionalcommits.org/) for commit messages:
  - `feat:` new feature
  - `fix:` bug fix
  - `docs:` documentation only
  - `chore:` misc changes

## Questions?
Open a [GitHub Discussion](../../discussions) or [issue](../../issues).
